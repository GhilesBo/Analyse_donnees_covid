Dans ce dossier vous trouverez:

-models/schema1.js : qui correspond au schema dans lesquelles sont organisé nos données dans la base et qui nous permettra de
manipuler les données

-pythonScript/ : dossier qui contient des fichiers pythons qui peuvent etre executés depuis app.js mais qui n'est plus utile depuis 
que l'in utilise plotly

-app.js: contient toute les methodes permettant de recuperer les données depuis des fichier python u bien la base données

-server.js: fichier principale qui demarre un serveur nodejs et qui s'assure que tout fonctionne avec un message d'erreur.
